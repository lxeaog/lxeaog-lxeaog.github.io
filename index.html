使用hexo+github page搭建个人博客

Posted on 2025-01-24 Edited on 2025-01-25 In 技术

前言

此前看了一些大佬的Powered by Hexo个人博客，想起了以前未曾完成的一个想法，搭建个人博客，记录一些感想、随记、工作记录，亦或者人生轨迹。也是作为2025的一个新起点。
Hexo 是一个快速、简洁且高效的博客框架，它基于 Node.js，可以将 Markdown 文件快速生成静态网页。GitHub Pages 是 GitHub 提供的免费静态网页托管服务，非常适合用来托管个人博客。
之所以选择Github Pages，主要还是因为部署方便。

在开始之前，确保你的电脑或者云服务器上已经安装了以下工具：

  • Node.js（建议使用 LTS 版本）
  • Git
